blip.blip.capture_callback(destination, header, content)[source]¶

Function which will be called on each packet capture which matches the filters.

It is meant to call other processing functions to transform packets into the right binary datastructures before writing them to output.

Parameters:
  • destination – File-like object to which the output is written
  • header – pcap capture header
  • content – pcap capture data

None
